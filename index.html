Módulos de Aprendizado

Módulo 1: Introdução à Lógica

Título: Variável / Tipos de Dados / Operadores Matemáticos

Exercícios do Módulo

  • Procura entender o que viu nos dois vídeos e depois faça um Resumo dos dois Videos. O resumo pode ser feito onde achares melhor.
  • OBS: É proibido usar Inteligencia Artificial para resumir os vídeos, caso o fizer não vai aprender nada e nuna vai conseguir programar na vida. Como iniciante deve aprender a fazer as tarefas sozinho para melhorar as suas habilidades na programação.

Módulo 2: Variáveis e Tipos de Dados

Exercícios do Módulo

  • Defina o conceito de variável na programação.
  • Quais são os tipos de dados mais comuns em programação?
  • Crie variáveis para armazenar: nome, idade, altura e se é estudante.
  • Explique a diferença entre os operadores =, == e ===.

Módulo 3: Estruturas Condicionais

Exercícios do Módulo

  • O que são estruturas condicionais e para que servem?
  • Descreva a diferença entre if, else if e else.
  • Crie um algoritmo que verifique se um número é positivo, negativo ou zero.
  • Escreva uma condicional que verifique se uma pessoa pode votar (idade >= 16).

Módulo 4: Estruturas de Repetição

Exercícios do Módulo

  • Qual a diferença entre os laços for, while e do-while?
  • Quando usar cada tipo de estrutura de repetição?
  • Crie um laço for que imprima os números de 1 a 10.
  • Escreva um algoritmo que some todos os números pares de 1 a 100 usando while.
